Purpose

HTTP Requests are composed of the following list of items in order:

  1. This; the first line of the request, will be different for a GET request vs a POST request". For a GET request you would see something such as: GET /api-endpoint-url HTTP/1.1
  2. Header1: value1
  3. Header2: value2
  4. Header3: value3
  5. optional: message body goes here, like file contents or query data; it can be many lines long, or even binary data $&*%@!^$@

In Javascript with use the browsers built in XMLHttpRequest Object to compose each part of the http request and handle the response we get back from it.
